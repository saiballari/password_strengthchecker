# Password Strength Checker

## Project Description

Password Strength Checker is a Python-based cybersecurity tool designed to evaluate the strength of a user's password. The program analyzes the password based on several security criteria, including length, uppercase letters, lowercase letters, numbers, and special characters. It then provides a strength rating to help users create stronger and more secure passwords.

## Features

* Checks if the password is at least 8 characters long
* Detects uppercase letters (A-Z)
* Detects lowercase letters (a-z)
* Detects numeric digits (0-9)
* Detects special characters
* Rates password strength as:

  * Very Weak
  * Weak
  * Medium
  * Strong
  * Very Strong

## Technologies Used

* Python
* Regular Expressions (re module)

## How It Works

1. The user enters a password.
2. The program checks the password against five security criteria.
3. A score is assigned based on the criteria met.
4. The final score determines the password strength level.
5. The result is displayed to the user.
